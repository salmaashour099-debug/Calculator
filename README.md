⚙️ How It Works
Input Numbers

Users can enter numbers by clicking the calculator buttons.

Perform Operations

The calculator supports:

Addition (+)
Subtraction (-)
Multiplication (*)
Division (/)
Percentage (%)
Delete

The DEL button removes the last entered character.

Clear

The CLR button clears the entire expression.

Calculate

The equals (=) button evaluates the expression and displays the result.
